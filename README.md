# Garmin Vault-Boy Watch Face
A Fallout-inspired "Pip-Boy" style watch face for Garmin Fenix 8 Solar (MIP).

## Features
* **Animated Vault Boy:** 7-frame walking animation.
* **Vault-Tec UI:** Progress bars for Steps, Floors, and Solar Intensity.
* **MIP Optimized:** High-performance caching to prevent screen flickering.
* **Stats:** Heart Rate, Body Battery, Calories, and Weather.

## Hardware Compatibility
* Optimized for **Garmin Fenix 8 Solar (MIP)**.
* Resolution: 280x280.
